== Mpower

=== Beschreibung
Ce plugin permet de contrôler vos multiprises Mpower. Le widget permet de voir toutes les infos :

Etat
Puissance
Consommation
Courant
Voltage
et Facteur de puissance

Les données sont rafraîchies toutes les minutes et après une action depuis jeedom
